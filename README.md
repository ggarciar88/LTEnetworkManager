# LTE Network Manager on Java
____________________________________________________________________________________________________________________________________________________________________________________________

## Description

This is a simplified LTE (or 5G) network management desktop application written in Java using Swing.
The project allows you to:

- Add new cells to the network
- Specify location, frequency, and bandwidth
- View a table of all active cells

This kind of tool is useful in telecom engineering to visualize and manage base stations or gNodeBs in small-scale
 simulated networks.

## Features

- Java Swing GUI
- Table display of network cells
- Basic data entry for cell configuration

## How to Run

1. Make sure you have Java (JDK 8+) installed.
2. Compile the program with the following sentence on the bash:

      javac Main.java NetworkManagerGUI.java

3. Run it:

Use sentence  ->  java Main

## Future Improvements

- Data persistence using SQLite or file I/O
- Frequency conflict detection
- Map integration using coordinates
- 5G NR-specific parameters
